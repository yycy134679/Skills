# 合规护栏（增强版）

## 1）目标

在保证转化能力的同时保持表达安全：移除高风险表述（绝对化、医疗化、保证性承诺），改为透明、可验证、可落地的话术。

## 2）风险等级

- `high`：必须替换或删除。
- `medium`：建议替换为更保守、限定条件的表达。
- `low`：可保留，但要有上下文证据支撑。

## 3）限制/禁用表达

| 风险 | 模式 / 示例 | 建议替换 |
|---|---|---|
| high | “100%有效”“保证见效”“guaranteed results” | “可能有帮助”“可支持日常场景”“部分用户反馈…” |
| high | “治愈”“治疗疾病”“medical-grade cure” | “提升使用体验”“帮助改善流程感受” |
| high | “所有人都适用”“零副作用”“works for everyone” | “因人而异”“建议先做局部测试/首次小范围试用” |
| high | “一周瘦几斤”“快速减肥”“躺着瘦” | “更适合日常管理场景”“饮食与体验因人而异” |
| high | “增强免疫力保证有效”“吃了就不失眠”“一吃就通便” | “用于日常营养补充/作息支持的场景表达，不承诺确定结果” |
| high | “宝宝绝对安全”“新生儿闭眼用”“可替代看护” | “强调需按说明使用，并结合具体阶段判断” |
| high | “所有猫狗都爱”“每只宠物都适应”“绝不应激” | “以当前宠物真实反馈为准，不替所有宠物下结论” |
| high | “绝对防滑”“绝对防摔”“绝对不脱落”“保证行车安全” | “在本次演示中更稳/更顺手，实际使用请按场景判断” |
| high | “承重无上限”“绝不坏”“终身不坏” | “按当前演示场景表现稳定，具体承重/耐用需看实际条件” |
| medium | “全网第一”“行业第一”“best on market” | “常见选择”“在某类场景下更常被偏好” |

| medium | “谁穿都显瘦”“任何家庭都适合”“所有设备都兼容” | “更适合某类身材/空间/设备，建议先对照自身情况判断” |
| medium | “用了成绩就会提高”“玩了就更聪明”“戴了运动表现一定更强” | “更适合某类学习/玩耍/训练场景，不承诺确定结果” |
| low | “很好用”“amazing”“great” | 如有真实演示证据可保留，否则弱化为中性描述 |

## 4）自动替换优先级

1. 先处理 `high` 风险短语。
2. 再处理 `medium` 风险短语。
3. `low` 风险仅在邻近有证据时保留。
4. 若无法安全替换，则直接删除该高风险表达。

## 5）软性 CTA 规则

使用建议式、低压力引导：

- “如果和你的日常场景匹配，可以考虑先试一次。”
- “先在一个真实场景测试，再对比结果。”
- “先收藏这条流程，等需要时照着做。”

避免高压催促：

- “现在不买就亏了”
- “最后机会人人必买”
- “今天必须下单”

## 6）类目加严提醒

- `health-wellness`：禁止疾病治疗、医学替代、精确减重、睡眠/免疫/肠胃确定性承诺。
- `mom-baby`：禁止绝对安全、替代监护、保证不哭不闹、不呛不漏等表述。
- `pet`：禁止承诺所有宠物都适应、都会喜欢、绝不应激。
- `beauty-skincare`：禁止医疗化改善、立刻换肤、永久祛痘祛斑等表述。
- `consumer-electronics` / `home-appliance`：禁止编造精确性能、兼容性、耐久性、安全性数据。
- `sports-outdoor`：禁止保证减脂、增肌、运动表现提升或绝对防护。
- `automotive-accessories` / `tools-hardware`：禁止绝对安全、防事故、无限承重、零风险安装等表述。
- `office-school` / `toys-games`：禁止承诺成绩提升、智力提升、专注力确定改善。
- `jewelry-watches`：禁止虚构材质、真伪、保值、绝对防水防刮或走时精度。
- `luggage-travel`：禁止保证所有航司都能登机、绝对防盗、绝对防水。
- `holiday-party`：涉及灯串、电池、明火或儿童派对用品时，避免绝对安全和材质无风险承诺。

## 7）合规说明模块要求

输出中必须包含 `合规说明`，并给出：

- `已替换高风险表达`：列出替换或删除的词句。
- `风险提醒`：给出简洁提醒（如“体验因人而异”“避免医疗承诺”）。

如果本次无需替换，写：

- `已替换高风险表达：无`
- `风险提醒：未检测到高风险承诺表达`
