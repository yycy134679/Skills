# 策略：保健滋补与营养补充

## 类目任务

- 核心成交逻辑：卖的不是“神奇效果”，而是“更容易坚持的日常补充方式”和“透明、安心的基础信息”。
- 更容易拿曝光的开场：从真实作息或饮食习惯切入，例如经常忘记补充、怕吞大颗、味道难接受、外出不方便。
- 最容易促成转化的证据：剂型、口味、食用/饮用便利度、成分和规格透明度、日常坚持成本、收纳与携带。

## 曝光优先切入

- 开场讲“我为什么能坚持用它”，比讲功效更容易建立信任。
- 如果是软糖、粉剂、冲剂，优先拍入口/冲调体验。
- 把“适合怎样的日常节奏”说清楚，让观众先代入生活方式。

## 转化优先证据

- 拍出剂型大小、口味反应、溶解状态、包装便携度。
- 近景展示成分表、每份规格、食用方式，但不要堆术语。
- 说明适合放在什么日常节点里，例如早餐后、运动后、通勤包里。
- 重点回答“好不好坚持、方不方便、口感能不能接受”。

## 达人拍法要求

- 人设更像“分享自己如何把补充动作融入生活”，不要像医疗科普或医生背书。
- 画面以桌面、厨房、包内、办公位等日常场景为主，降低距离感。
- 节奏上先讲为什么会坚持，再讲产品如何降低门槛。
- 口播保持克制，避免说“吃了就会怎样”，多说“对我来说更容易持续”。

## 最佳 Hook 模式

- “我现在会留下来的补充品，标准只有一个：能不能真的坚持下去。”
- “如果你也很怕难喝、难吞、记不住，这种形式会更容易进日常。”
- “我选它不是因为神奇，是因为它把这件事做得足够省心。”

## 推荐分镜推进

1. 用真实作息痛点或坚持难题开场。
2. 展示包装、剂型或冲调准备。
3. 展示入口/冲调/食用体验。
4. 近景交代核心成分或规格信息。
5. 说明日常放在哪里、什么时候使用更顺手。
6. 回答口味、大小、刺激感、性价比等疑问。
7. 用“适合哪类人先试”做软性收尾。

## 常见异议与回应

| 异议 | 回应模式 |
|---|---|
| “会不会难坚持？” | “从口味、剂型、携带和使用时机来回应。” |
| “会不会太甜/太腥/难喝？” | “用真实口感描述，不做夸张承诺。” |
| “值不值？” | “说明日常使用频率和每次补充的便利性。” |
| “适不适合我？” | “只讲一般适配场景，不给个体化功效判断。” |

## 对比规则

```yaml
comparison_recommended: false
comparison_when:
  - 仅在剂型大小、口感接受度、冲调便捷度差异可真实展示时使用
comparison_type:
  - 旧补充方式_vs_更易坚持的方式
```

## 软性 CTA 模式

- “如果你正在找更容易坚持的补充方式，可以先从这种形式试起。”
- “先看成分和剂型是不是你能接受的，再决定要不要加入日常。”
- “先收藏这个吃法/用法思路，回头对照自己的作息会更直观。”

## 建议 / 避免

| 建议 | 避免 |
|---|---|
| 重点讲坚持成本和日常便利。 | 把营养补充说成治病方案。 |
| 透明展示成分、规格、剂型。 | 宣称立刻见效或人人适用。 |
| 用生活方式语言表达体验。 | 伪造身体变化或医疗对比。 |

## 类目专属合规提醒

- 严禁疾病治疗、治愈、医学替代、绝对功效承诺。
- 睡眠、免疫、消化、关节等表述要保守，最多描述日常支持或使用体验。
