# 策略：美妆个护

## 类目任务

- 核心成交逻辑：让用户快速看到“上脸/上发/上身后的真实变化”，再用肤感、使用门槛和适配人群降低犹豫。
- 更容易拿曝光的开场：先给妆效差异、质地触感、毛躁油塌、口气焦虑、身体护理前后等熟悉问题，不要先讲复杂术语。
- 最容易促成转化的证据：质地、延展性、泡沫/喷雾状态、使用前后观感、气味方向、流程是否省事。

## 曝光优先切入

- 在前 3 秒让人和产品同时出现在真实护理场景里，建立“这跟我有关”的代入感。
- 优先拍暗沉、卡粉、出油、炸毛、干燥、口腔护理不便等高频小问题。
- 如果产品的质地、泡沫、上妆效果很有辨识度，用微距和侧光做第一波停留。

## 转化优先证据

- 展示真实肤感、发感或使用后的细节变化，例如服帖、顺滑、清爽、柔润、便捷。
- 优先用自然光或真实室内光，减少滤镜和磨皮带来的不信任。
- 明确说明适合的肤质、发质、使用顺序和使用时机。
- 给出用量、单次使用范围和坚持成本，帮助用户判断值不值。

## 达人拍法要求

- 人设优先“真实自用分享”，少用品牌宣讲腔。
- 画面以面部近景、发丝近景、手部质地近景、局部前后对比为主。
- 节奏上先看问题，再看使用，再看结果，避免直接念卖点。
- 口播多用感受型、场景型语言，少堆成分和术语。

## 最佳 Hook 模式

- “我不是又来推荐一个瓶子，是这个细节真的把我打动了。”
- “如果你每天都卡在这个护理步骤，先看它到底有没有更省事。”
- “我最怕那种看起来厉害、用起来麻烦的，这个倒不是。”

## 推荐分镜推进

1. 用皮肤、头发或护理流程中的真实痛点开场。
2. 近景展示产品质地、喷头、膏体、刷头或使用形态。
3. 展示真实上脸、上发、清洁或护理过程。
4. 补使用后的观感、触感、服帖度或流程便利。
5. 说明适合人群、使用顺序和高频场景。
6. 回答黏不黏、刺激吗、值不值、适不适合我的疑问。
7. 以小范围试用或先收藏流程的方式收尾。

## 常见异议与回应

| 异议 | 回应模式 |
|---|---|
| “会不会很黏/很厚重？” | “镜头里展示质地铺开、吸收或成膜后的真实观感。” |
| “会不会刺激/不适合我？” | “说明适用人群与建议测试方式，不做绝对安全承诺。” |
| “值不值这个价？” | “换算成单次使用价值与流程省时价值。” |
| “是不是又要多加一步？” | “用连续镜头证明上手门槛低、不会拖长流程。” |

## 对比规则

```yaml
comparison_recommended: true
comparison_when:
  - 质地、妆效、顺滑度或使用流程差异可被清晰拍到
  - 可在不造假前提下呈现前后差异
comparison_type:
  - 本品_vs_旧流程
  - 使用前_vs_使用后
```

## 软性 CTA 模式

- “如果你的情况和我接近，可以先从一次小范围试用开始。”
- “先收藏这套使用顺序，回头照着试会更直观。”
- “先看你在不在这个人群里，再决定要不要入。”

## 建议 / 避免

| 建议 | 避免 |
|---|---|
| 在真实光线下展示细节变化。 | 宣称立刻治愈、永久改善。 |
| 说清楚适合谁、不适合谁。 | 使用“所有人都 100% 安全”这类绝对化表达。 |
| 用流程价值讲清购买理由。 | 过度依赖滤镜和修图制造反差。 |

## 类目专属合规提醒

- 避免把美妆、护肤、洗护体验说成医疗效果或治疗结果。
- 不要承诺零刺激、永久改善、人人适用。
