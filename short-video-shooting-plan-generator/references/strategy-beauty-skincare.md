# 策略：美妆与护肤

## 类目任务

- 核心成交逻辑：让用户快速看到“上脸/上妆后的真实变化”，再用肤质适配和使用场景降低犹豫。
- 更容易拿曝光的开场：先给皮肤状态、妆效差异或质地触感，不要先讲复杂成分。
- 最容易促成转化的证据：质地、延展性、吸收/服帖感、妆前妆后状态、持妆或日常流程适配。

## 曝光优先切入

- 在前 3 秒让脸和产品同时出现，快速建立“这跟我有关”的代入感。
- 优先拍“暗沉、卡粉、厚重、拔干、油光”等用户熟悉问题。
- 如果质地很有辨识度，用微距和侧光做第一波停留。

## 转化优先证据

- 展示真实肤感变化，例如水润感、清爽感、服帖度或后续上妆状态。
- 优先用自然光或真实室内光，减少滤镜和磨皮带来的不信任。
- 明确说明适合的肤质、使用顺序和使用时机。
- 给出用量、延展面积和单次使用感受，帮助用户判断值不值。

## 达人拍法要求

- 人设优先“真实自用分享”，少用过度完美的商业片口吻。
- 画面以面部近景、手部质地近景、半脸对比、妆前妆后切换为主。
- 节奏上先看问题，再看上脸，再看结果，避免直接念卖点。
- 口播多用感受型、场景型语言，少堆术语和成分教科书。

## 最佳 Hook 模式

- “如果你到中午就容易暗沉，先看这个 10 秒妆前小步骤。”
- “我连续测了 7 天，最先变化的是这个细节。”
- “不是又一款厚重面霜，看看它在真实皮肤上的质地。”

## 推荐分镜推进

1. 用面部痛点、妆面问题或结果预告开场。
2. 近景展示质地、颜色、延展性。
3. 展示真实上脸/上妆过程。
4. 补吸收、服帖、后续妆效或肤感结果。
5. 给出使用顺序和适合人群。
6. 回答黏不黏、闷不闷、值不值等疑问。
7. 以小范围试用或先收藏流程的方式收尾。

## 常见异议与回应

| 异议 | 回应模式 |
|---|---|
| “会不会很黏？” | “镜头里展示快速吸收，并说明成膜观感（水润/清爽）。” |
| “会不会闷痘？” | “避免医疗承诺；建议先做局部测试，并强调成分透明。” |
| “值不值这个价？” | “换算成单次使用价值与流程省时价值。” |
| “适合我这种肤质吗？” | “明确说清肤质和使用时机，不要假装适合所有人。” |

## 对比规则

```yaml
comparison_recommended: true
comparison_when:
  - 质地或妆效差异可被清晰拍到
  - 可在不造假前提下呈现前后差异
comparison_type:
  - 本品_vs_旧流程
  - 本品_vs_常见痛点方案
```

## 软性 CTA 模式

- “如果你的肤质和我接近，这款可以先试一次。”
- “可以先从小规格开始，这周感受一下皮肤状态。”
- “先收藏这套使用顺序，今晚照着试试。”

## 建议 / 避免

| 建议 | 避免 |
|---|---|
| 在自然光下展示真实皮肤质感。 | 宣称立刻治愈或永久见效。 |
| 说清楚适合谁、不适合谁。 | 使用“所有人都 100% 安全”这类绝对化表达。 |
| 用“流程价值”讲清购买理由。 | 承诺医疗治疗结果。 |

## 类目专属合规提醒

- 避免把护肤、美妆体验说成医疗效果或皮肤问题治疗。
- 不要用重滤镜、重修图制造不真实前后反差。
