# 策略：家居装饰 & 节庆装饰

## 类目任务

- 核心成交逻辑：卖的是“氛围感”和“完成度”，先让用户看到空间变化，再证明布置或摆放并不麻烦。
- 更容易拿曝光的开场：前后反差、点亮瞬间、摆上后的空间氛围、拍照成片效果，都比静态讲解更抓眼。
- 最容易促成转化的证据：布置前后变化、摆放难度、空间适配、质感细节、日常/节庆双场景价值。

## 曝光优先切入

- 第一秒直接给布置完成后的效果，或给“没摆之前很普通”的反差。
- 如果是节庆装饰，明确生日、节日、聚会、节庆换装等场景。
- 如果是日常摆件或软装，优先拍它放进空间后的整体氛围，而不是孤立白底图。

## 转化优先证据

- 拍一次真实摆放、悬挂、点亮或组合过程，证明不是很难实现。
- 展示颜色、层次、灯光、材质和空间比例，体现成片效果。
- 说明适合的空间大小、摆放区域和使用周期。
- 回答“麻不麻烦”“会不会显廉价”“就用一次值不值”的疑问。

## 达人拍法要求

- 人设优先生活方式/家居达人，而不是活动策划说明书。
- 画面以布置前后、中景环境、局部细节、人物互动或拍照画面为主。
- 节奏上先给结果惊喜，再补摆放方法。
- 口播多说“出片、氛围、准备成本低”，少说空泛浪漫形容词。

## 最佳 Hook 模式

- “这个角落我一直觉得差点意思，摆上它之后终于完整了。”
- “如果你想用最少时间把氛围做出来，先看这个前后反差。”
- “它不是那种摆一次就算了的装饰，日常拍照和节庆都挺有用。”

## 推荐分镜推进

1. 用空间前后反差或成品氛围开场。
2. 展示产品形态、套装内容或摆放位置。
3. 连续拍一次摆放、上墙、点亮或组合过程。
4. 展示整体环境效果和局部细节。
5. 给出拍照、聚会、居家角落美化等真实场景。
6. 回答安装难度、空间适配、一次性还是可复用等问题。
7. 用“适合哪类空间先试”做软性收尾。

## 常见异议与回应

| 异议 | 回应模式 |
|---|---|
| “布置会不会很麻烦？” | “用连续镜头展示实际步骤和时间感。” |
| “会不会看起来廉价？” | “拍局部细节和整体成片效果。” |
| “就用一次值不值？” | “说明可复用、留影价值或日常场景适配。” |
| “家里空间够吗？” | “给空间参照和摆放区域建议。” |

## 对比规则

```yaml
comparison_recommended: true
comparison_when:
  - 布置前后氛围差异明显
  - 普通空间和完成布置后的反差清晰
comparison_type:
  - 未布置_vs_布置完成
  - 空角落_vs_摆放后
```

## 软性 CTA 模式

- “如果你也想把这个空间做得更完整，可以先从这类装饰试起。”
- “先看你的空间和预算适不适合，再决定要不要入。”
- “先收藏这套布置思路，等活动前或改造角落时照着抄。”

## 建议 / 避免

| 建议 | 避免 |
|---|---|
| 优先拍前后反差和成片效果。 | 只讲梦幻感却不拍环境变化。 |
| 把布置难度讲清楚。 | 虚构绝对安全、永不掉落、永不漏气。 |
| 明确节庆或空间场景。 | 把一次性用品硬说成永久耐用。 |

## 类目专属合规提醒

- 涉及灯串、电池、明火、玻璃或儿童节庆用品时，不要做绝对安全承诺。
- 不要虚构材质等级、阻燃等级或极限耐久表现。
