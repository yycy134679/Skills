# 策略：个护与清洁

## 类目任务

- 核心成交逻辑：在真实清洁/护理任务里证明“更省事、感受更好、结果更直观”。
- 更容易拿曝光的开场：脏污、异味、毛躁、头皮油塌、清洁死角、早晚护理麻烦等真实问题，都是高停留入口。
- 最容易促成转化的证据：前后变化、泡沫/清洁过程、气味和肤感描述、使用门槛、适用材质或部位。

## 曝光优先切入

- 开场最好直接给问题现场，而不是先举着产品解释。
- 如果结果看得见，用前后区域对比抓停留。
- 如果是个护用品，优先拍使用前后的状态变化，而不是只拍包装。

## 转化优先证据

- 连续展示一次完整使用，不要靠剪辑制造“突然变干净/变顺滑”。
- 说清适用场景、适用材质、适用发质/肤感/气味偏好。
- 把省时、省力、气味感受或洗后状态拍具体。
- 如有喷头、刷头、一次用量等细节，尽量拍出真实手感。

## 达人拍法要求

- 人设以“自己真的在做这件清洁/护理任务”为主。
- 画面以问题现场、手部操作、结果近景为核心，不要全程干讲。
- 节奏上先给脏污/不适感，再给使用，再给结果和维护。
- 口播尽量把感受讲清楚，例如清爽、顺、好冲洗、没那么刺鼻。

## 最佳 Hook 模式

- “我把清洁时间缩短一半，靠的是这个方法。”
- “你明明清理了但还是有异味？先看这个。”
- “一个小替换，让我早晨流程轻松很多。”

## 推荐分镜推进

1. 以常见脏乱或不便场景开场。
2. 一步展示产品准备就绪。
3. 直接在目标区域使用。
4. 展示可见结果与节省时间。
5. 加一个适配人群提示（小户型、忙碌作息、气味敏感等）。
6. 用低摩擦下一步动作收尾。

## 常见异议与回应

| 异议 | 回应模式 |
|---|---|
| “真的更干净吗？” | “在同等条件下展示对比区域差异。” |
| “味道会不会太冲？” | “用日常语描述气味层次和适用场景。” |
| “会不会伤表面/刺激皮肤？” | “说明适用材质或部位，并建议先小范围测试。” |
| “会不会很麻烦？” | “用连续镜头证明准备和收尾都不复杂。” |

## 对比规则

```yaml
comparison_recommended: true
comparison_when:
  - 可见清洁差异存在
  - 可测量节省时间
comparison_type:
  - 同一表面_前后对比
  - 手工方式_vs_产品方式
```

## 软性 CTA 模式

- “如果你想把日常清洁做得更省力，这个方案值得先试。”
- “先在一个高频区域试一次，再对比你自己的结果。”
- “先收藏这套方法，下一次清洁日直接照做。”

## 建议 / 避免

| 建议 | 避免 |
|---|---|
| 从头到尾展示一个真实使用场景。 | 过度剪辑制造“假清洁效果”。 |
| 说明最适用的材质/区域。 | 宣称“所有材质都零风险适用”。 |
| 强调省事和流程适配。 | 用恐惧式话术逼单。 |

## 类目专属合规提醒

- 不要把清洁、除味、护理表达成消毒治病或绝对杀灭。
- 个护产品避免承诺永久改善、零刺激、人人适用。
