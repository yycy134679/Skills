# 策略：珠宝饰品与手表

## 类目任务

- 核心成交逻辑：先拍“戴上去是否有气质和存在感”，再用质感细节、尺寸比例和搭配场景增强转化。
- 更容易拿曝光的开场：第一秒给上手、上耳、上颈的佩戴效果，或直接给穿搭完成后的整体气质变化。
- 最容易促成转化的证据：光泽、比例、厚薄、细节做工、扣合方式、搭配场景和佩戴舒适度。

## 曝光优先切入

- 优先拍佩戴效果，不要先在桌面平铺聊材质。
- 用不同肤色/穿搭背景下的反光和存在感抓停留。
- 手表类可以先给抬手看时间、敲键盘、通勤穿搭这种生活化动作。

## 转化优先证据

- 拍近景细节，例如链节、表盘、刻度、扣头、耳针、镶嵌或纹理。
- 给尺寸参照，说明大小、贴合度、戴上后的份量感。
- 展示至少一个日常搭配和一个稍正式场景，降低想象门槛。
- 回答“会不会太夸张”“会不会压手/硌”“好不好搭”“值不值”。

## 达人拍法要求

- 人设优先穿搭/生活方式分享，而不是柜台导购式介绍。
- 画面以佩戴中景、局部细节特写、整体搭配效果交替。
- 节奏上先看气质提升，再看细节，再看搭配场景。
- 口播要说“显精致、提气色、搭衣服更完整”这类可感知结果，少说空泛高级感。

## 最佳 Hook 模式

- “这种小配饰最怕看着好看，戴上去却没存在感，这个我上手后还挺惊喜。”
- “如果你想让一套基础穿搭看起来更完整，这种细节会很加分。”
- “我会留下这只表，不是因为它贵气，而是通勤戴起来真的很顺眼。”

## 推荐分镜推进

1. 用佩戴效果或整体搭配完成后的状态开场。
2. 展示产品外观和上手/上耳/上颈动作。
3. 补细节近景，例如光泽、纹理、扣合。
4. 展示第二套搭配或第二个场景。
5. 说明尺寸、厚薄、重量感或表带调节方式。
6. 回答好不好搭、会不会夸张、佩戴舒不舒服。
7. 用“适合哪类风格/送礼场景”做软性收尾。

## 常见异议与回应

| 异议 | 回应模式 |
|---|---|
| “会不会太夸张/太难搭？” | “用基础穿搭和正式穿搭各展示一次。” |
| “会不会显廉价？” | “近景拍纹理、反光和做工细节。” |
| “戴着舒服吗？” | “说明重量感、贴肤感和调节方式。” |
| “值不值？” | “强调佩戴频率和搭配完成度，而不是空喊材质。” |

## 对比规则

```yaml
comparison_recommended: true
comparison_when:
  - 佩戴前后气质变化或搭配完成度差异明显
  - 同一套穿搭在有无配饰时差异清晰
comparison_type:
  - 无配饰_vs_佩戴后
  - 基础穿搭_vs_搭配完成后
```

## 软性 CTA 模式

- “如果你平时就喜欢这种低调但提气质的配饰，可以先从这类款式试起。”
- “先看你的穿搭风格和佩戴习惯匹不匹配，再决定要不要入。”
- “先收藏这套搭配思路，回头照着配会更直观。”

## 建议 / 避免

| 建议 | 避免 |
|---|---|
| 先拍佩戴效果，再讲细节。 | 全程桌面平铺不让人代入。 |
| 用搭配场景降低想象成本。 | 虚构材质、真伪、保值或绝对防水。 |
| 给尺寸和重量感参照。 | 只说“高级”“显贵”不说为什么。 |

## 类目专属合规提醒

- 不要虚构贵金属、宝石真伪、保值能力、绝对防水防刮或精确走时能力。
- 如无证据，避免承诺“不掉色”“不过敏”“戴多久都像新的一样”。
