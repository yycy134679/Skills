# 策略：居家日用

## 类目任务

- 核心成交逻辑：把“家里每天都会遇到的小摩擦”拍出来，让用户看到一件日用品如何换来更顺手、更省心的日常。
- 更容易拿曝光的开场：先给凌乱、难找、难放、难清洁、耗材更换麻烦、容易积灰等高频痛点。
- 最容易促成转化的证据：前后整洁度变化、收纳效率、清洁效率、容量和尺寸参照、补充更换成本。

## 曝光优先切入

- 第一秒优先给“乱”和“堵”的状态，再给解决后的顺滑状态。
- 用真实家庭角落、卫生间、玄关、洗衣区、柜内场景，而不是纯样板间。
- 如果是清洁或耗材类，前 3 秒要让用户看见“省哪一步”“少折腾哪一步”。

## 转化优先证据

- 拍清楚安装、抽拉、折叠、替换、擦拭、收纳等关键动作。
- 给尺寸参照和容量参照，帮助用户判断能不能放进自己家。
- 用前后对比展示动线是否更顺、台面是否更整齐、清洁是否更快。
- 补充维护、补货频率和复用场景，提升说服力。

## 达人拍法要求

- 人设优先“真实居家使用者”，用生活流表达比样板房展示更容易转化。
- 画面可用俯拍、中景和局部细节切换，重点拍动作变化和整理效果。
- 节奏上先解决问题，再展示附加价值，如省空间、好拿、好清理。
- 口播要具体指出省了哪一步、腾出了哪块空间、少做了什么重复动作。

## 最佳 Hook 模式

- “这个东西看起来普通，但它把我家最烦的那一步省掉了。”
- “如果你家也总卡在这个小问题，先看它有没有更顺。”
- “不是大改造，就是一个小替换，日常差别还挺明显。”

## 推荐分镜推进

1. 先展示家中真实摩擦点或清洁/收纳痛点。
2. 展示产品摆放、安装或替换过程。
3. 展示即时功能改进或清洁结果。
4. 展示第二使用场景或补充动作，体现复用价值。
5. 展示维护、补货、擦拭或节省空间细节。
6. 说明哪些人、哪些家庭场景适合直接照抄。
7. 用“先从一个区域试起”的软性动作收尾。

## 常见异议与回应

| 异议 | 回应模式 |
|---|---|
| “我真的需要吗？” | “关联到每天重复出现的痛点和省力价值。” |
| “我家放得下吗？” | “镜头里给尺寸与空间参照。” |
| “会不会难维护？” | “几秒内展示补充、擦拭、替换或收纳动作。” |
| “容量/数量真的够吗？” | “用实物装入或真实使用次数代替空口描述。” |

## 对比规则

```yaml
comparison_recommended: true
comparison_when:
  - 前后空间、整洁度或清洁效率提升可见
  - 可比较重复性日常摩擦
comparison_type:
  - 杂乱_vs_有序
  - 旧流程_vs_优化后流程
```

## 软性 CTA 模式

- “如果你家也是这个状态，可以照着这套摆法先试一个区域。”
- “先在一个高频角落试试看，再决定要不要全面换。”
- “收藏这条，等下次整理或补货时直接照着做。”

## 建议 / 避免

| 建议 | 避免 |
|---|---|
| 使用真实家庭场景，而不是纯样板间镜头。 | 无证据夸大容量、抑菌或耐久。 |
| 给出一个明确摆放或清洁技巧。 | 一条视频塞太多无关产品。 |
| 语言保持实用、可执行。 | 把普通日用品说成“改变人生”。 |

## 类目专属合规提醒

- 不要虚构承重、容量、抑菌、除味、全材质适配等结论。
- 清洁与纸品类避免写成绝对消毒、零残留、人人适用。
