# 策略：家居与日用

## 类目任务

- 核心成交逻辑：把“家里每天都会遇到的小摩擦”拍出来，让用户看到一件小东西如何换来更顺手的日常。
- 更容易拿曝光的开场：先给凌乱、占地、难找、难放、难拿、容易积灰等居家痛点。
- 最容易促成转化的证据：收纳前后、空间利用变化、安装难度、容量和尺寸参照、清洁维护成本。

## 曝光优先切入

- 第一秒优先给“乱”和“堵”的状态，再给解决后的顺滑状态。
- 用家庭真实角落而不是样板间，观众更容易代入。
- 如果是一物多用或小空间救星，前 3 秒要把“省空间”拍明白。

## 转化优先证据

- 拍清楚安装、摆放、抽拉、折叠、拿取等关键动作。
- 给尺寸参照和容量参照，帮助用户判断能不能放进自己家。
- 用前后对比展示动线是否更顺、台面是否更整齐。
- 补充清洁维护、耐用和复用场景，提升转化说服力。

## 达人拍法要求

- 人设优先“真实居家使用者”，用生活流表达比“精致样板房展示”更容易转化。
- 画面可用俯拍、中景和局部细节切换，重点拍动线变化。
- 节奏上先解决问题，再展示附加价值，如好看、整齐、一物多用。
- 口播要具体指出省了哪一步、腾出了哪块空间。

## 最佳 Hook 模式

- “这个居家小技巧，我真希望早点知道。”
- “它把我家最乱的角落整理明白了。”
- “小东西，但日常动线差别很大。”

## 推荐分镜推进

1. 先展示家中真实凌乱点或使用摩擦点。
2. 展示产品摆放/安装过程。
3. 展示即时功能改进。
4. 展示第二使用场景，体现一物多用。
5. 展示清洁维护或节省空间细节。
6. 说明哪些人适合直接照抄。
7. 用“先从一个角落试起”的软性动作收尾。

## 常见异议与回应

| 异议 | 回应模式 |
|---|---|
| “我真的需要吗？” | “关联到每天重复出现的痛点和省力价值。” |
| “我家放得下吗？” | “镜头里给尺寸与空间参照。” |
| “会不会难维护？” | “几秒内展示擦拭/清洁/收纳动作。” |
| “容量真的够吗？” | “用实物装入过程代替空口描述。” |

## 对比规则

```yaml
comparison_recommended: true
comparison_when:
  - 前后空间或整洁度提升可见
  - 可比较重复性日常摩擦
comparison_type:
  - 杂乱_vs_有序
  - 旧布局_vs_新布局
```

## 软性 CTA 模式

- “如果你家也是这个状态，可以照着这套摆法先试一块区域。”
- “先从一个角落开始，感受日常流程有没有更顺。”
- “收藏这套布局思路，周末整理时直接用。”

## 建议 / 避免

| 建议 | 避免 |
|---|---|
| 使用真实家庭场景，而非纯样板间镜头。 | 无证据夸大收纳容量。 |
| 给出一个明确摆放技巧。 | 一条视频塞太多无关产品。 |
| 语言保持实用、可执行。 | 把小工具说成“改变人生”。 |

## 类目专属合规提醒

- 不要虚构承重、容量、材质耐久等结论。
- 尺寸相关表述尽量给参照，避免模糊承诺“任何家庭都放得下”。
