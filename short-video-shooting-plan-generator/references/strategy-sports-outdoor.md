# 策略：运动与户外

## 类目任务

- 核心成交逻辑：让用户看到“这个装备在真实动作里真的好用”，先证明性能和舒适度，再讲便携、收纳和适合谁。
- 更容易拿曝光的开场：第一秒就给动作场景或问题场景，例如打滑、硌手、收纳麻烦、户外携带不方便。
- 最容易促成转化的证据：真实动作表现、防滑/支撑/回弹/轻量化、折叠收纳、出汗后状态、场景适配。

## 曝光优先切入

- 前 3 秒优先展示动作结果，不要先站着介绍装备。
- 如果差异能拍出来，用“原来不顺手 vs 现在更顺”的对比抓停留。
- 运动类先拍稳定性和反馈感，户外类先拍携带和使用便利。

## 转化优先证据

- 至少拍一次完整使用动作，例如拉伸、跑跳、收折、固定、搭建。
- 用中近景拍材质、厚度、抓地、支撑点和收纳细节。
- 说明适用人群和场景，例如居家训练、通勤健身房、短途露营。
- 回答“会不会鸡肋”“会不会占地方”“新手能不能直接用”。

## 达人拍法要求

- 人设优先“真实训练/户外使用者”，不要只做站桩口播。
- 画面以全身动作、中景演示、局部材质细节交替，避免全程静态特写。
- 节奏上先证明它在动作里成立，再补充携带和收纳价值。
- 口播要说“哪里更稳、哪里更省力、哪里更方便”，少喊热血口号。

## 最佳 Hook 模式

- “如果你每次做到这个动作都觉得不稳，先看这个装备能不能把问题解决掉。”
- “我会一直留下这个，不是因为它看起来专业，而是因为真的更顺手。”
- “这个户外小装备最打动我的，不是参数，是它把麻烦步骤省掉了。”

## 推荐分镜推进

1. 用动作痛点或使用结果开场。
2. 展示产品形态和上手门槛。
3. 连续拍一次核心动作或使用过程。
4. 补材质、抓地、支撑、回弹或轻量化细节。
5. 展示收纳、折叠、携带或第二场景。
6. 回答新手好不好上手、值不值、会不会占地方。
7. 用“适合哪类训练/户外人群先试”做软性收尾。

## 常见异议与回应

| 异议 | 回应模式 |
|---|---|
| “真的稳吗？” | “通过真实动作和局部近景展示稳定性，不靠口头保证。” |
| “会不会占地方？” | “拍出折叠、卷起、背包收纳或角落摆放。” |
| “新手能用吗？” | “用最基础动作或最简单场景示范上手门槛。” |
| “值不值买？” | “绑定每周高频训练/出行场景，解释它省了哪一步。” |

## 对比规则

```yaml
comparison_recommended: true
comparison_when:
  - 动作稳定性、支撑感或收纳便利度差异清晰
  - 原有方案存在明显摩擦
comparison_type:
  - 原来做法_vs_使用装备后
  - 收纳前_vs_收纳后
```

## 软性 CTA 模式

- “如果你也卡在这个训练/出行场景，可以先从这个装备试起。”
- “先看你最常用的动作或场景匹不匹配，再决定要不要入。”
- “先收藏这套用法，回头照着测一下自己会更直观。”

## 建议 / 避免

| 建议 | 避免 |
|---|---|
| 在真实动作里拍证据。 | 只讲专业名词不展示结果。 |
| 让观众看见收纳和携带成本。 | 保证减脂、增肌或成绩提升。 |
| 明确适合的新手/进阶场景。 | 夸大防护或绝对安全。 |

## 类目专属合规提醒

- 不要承诺减脂、增肌、运动成绩或身体改变的确定结果。
- 护具、防滑、防护类产品避免绝对安全和零受伤承诺。
